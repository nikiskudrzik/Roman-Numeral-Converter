# Roman Numeral Converter

## Description
This program converts a Roman numeral into its integer value.  
It uses a class called romanType to store the Roman numeral and convert it.

## Inputs
- A Roman numeral entered by the user (example: MCXIV)
- A menu choice (1, 2, or 3)

## Outputs
- Roman numeral
- Integer value
- Or both, depending on the user's choice

## How to Compile and Run
Compile:
g++ main.cpp romanType.cpp -o romanConverter

Run (I use a Macbook Air):
./romanConverter
